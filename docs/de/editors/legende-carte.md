# Legende der Schweizer Karte

Die Kartenlegende beschreibt die in der Schweiz verwendeten Segmenttypen und wie sie im Waze-Editor (WME) dargestellt werden. Jeder Strassentyp hat eine bestimmte Verwendung und Auswirkungen auf das Routing.

::: note Im Aufbau
Die Details zu den Segmenttypen, ihren Farben und ihren Anwendungsfällen sind hier noch nicht erfasst. Vorerst ist diese Seite eine Einführung. Der vollständige Inhalt wird aus dem Schweizer Wiki ergänzt.
:::

## Wozu die Legende dient

Die Wahl des Segmenttyps beeinflusst die Führung direkt. Ein falscher Typ kann Fahrer auf ungeeignete Strassen schicken oder sie umgekehrt von gültigen Routen ausschliessen. Die Legende dient als gemeinsame Referenz, damit alle Editoren Strassen auf dieselbe Weise benennen und klassifizieren.
