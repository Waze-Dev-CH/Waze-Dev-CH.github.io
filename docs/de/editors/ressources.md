# Nützliche Ressourcen

Diese Seite fasst die externen Links und Werkzeuge zusammen, die für die Bearbeitung der Waze-Karte in der Schweiz nützlich sind. Im Zweifelsfall haben lokale Ressourcen (Schweizer Wiki, Forum, Discord) immer Vorrang vor Improvisation.

## Wikis und Wazeopedia

Das deutschsprachige Wiki ist die Referenz für die Bearbeitung in der Schweiz, da die französischen Übersetzungen im Rückstand sein können.

- Wazeopedia Schweiz: https://www.waze.com/discuss/c/wazeopedia/switzerland-wazeopedia/5228
- Französischsprachiger Hub (Informationen auf Französisch): https://www.waze.com/discuss/t/informations-en-francais/377254
- Deutschsprachiger Hub (Hauptseite): https://www.waze.com/discuss/t/hauptseite/377253

::: note Sprachen
Der französischsprachige Hub fasst die für die Westschweiz übersetzten Seiten zusammen. Die Hauptseite bleibt die Hauptseite auf der DACH-Seite.
:::

## Werkzeuge

### WME Switzerland Helper

Tampermonkey-Userscript für den Editor. Es fügt hinzu:

- Schweizer geografische Ebenen (Grenzen, Luftbilder usw.),
- Daten des öffentlichen Verkehrs (SBB-Haltestellen),
- eine Prüfung der Strassennamen.

::: note Notiz
Das Skript wird über einen Userscript-Manager wie Tampermonkey installiert.
:::
