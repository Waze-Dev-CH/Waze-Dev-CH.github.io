# Chemins piétons

Waze est une application de navigation automobile. Les chemins piétons ne se cartographient que lorsqu'ils servent réellement la navigation routière. Le reste encombre l'affichage sans bénéfice.

## Quand inclure un chemin piéton

Deux cas justifient de tracer un chemin piéton :

- il sert d'aide visuelle à la navigation pour le conducteur, en particulier quand le segment porte un nom connu
- il permet le routing vers une destination autrement inaccessible, en guidant le conducteur vers le bon point d'accès quand la destination ne se rejoint qu'à pied

## Quand ne pas inclure

À ne pas cartographier :

- les trottoirs le long des routes
- les sentiers cachés, invisibles depuis la route et repérables seulement sur les vues aériennes
- les cheminements internes aux parcs ou aux propriétés privées

::: important
Un chemin piéton ne se justifie que s'il améliore concrètement le routing ou l'orientation du conducteur. Dans le doute, on ne le trace pas.
:::
