# Dangers permanents

::: warning Niveau L4+
La configuration des dangers permanents est réservée aux éditeurs de niveau 4 et plus.
:::

Les **dangers permanents** permettent de signaler des situations dangereuses fixes sur la carte (passages à niveau, virages à visibilité réduite, zones accidentogènes, etc.).

::: note Mise à jour — juin 2024
Les règles pour les **passages à niveau** ont été intégrées dans cette section. Elles étaient auparavant dans les guidelines Lanes (L3+).
:::

## Accès dans le WME

La fonctionnalité est accessible dans le WME aux éditeurs L4+. Si le segment concerné est verrouillé à un niveau supérieur, un déverrouillage sera nécessaire.
