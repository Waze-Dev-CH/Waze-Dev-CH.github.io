# Vignette autoroutière suisse

::: important
La vignette est **obligatoire** sur toutes les autoroutes suisses. L'amende en cas d'absence est de **CHF 200**.
:::

La vignette autoroutière suisse coûte **CHF 40 par an** et doit être apposée sur le pare-brise du véhicule. Sans confirmation dans Waze, l'application évite automatiquement les autoroutes lors du calcul d'itinéraire.

## Activer le routage autoroutier dans Waze

1. Ouvrir **Paramètres** dans l'application Waze
2. Aller dans **Navigation**
3. Sélectionner **Ajouter un pass péage/covoiturage**
4. Choisir **Vignette CH** et confirmer

Une fois activé, Waze utilisera les autoroutes suisses dans ses calculs d'itinéraire.

::: warning
Modifier ce paramètre dans Waze **ne remplace pas l'obligation d'acheter la vignette**. L'achat doit être effectué séparément via les canaux officiels.
:::

## Acheter la vignette

- [e-vignette.ch](https://e-vignette.ch/) — vignette numérique officielle
- [via.admin.ch/shop](https://via.admin.ch/shop) — boutique officielle de la Confédération

## Tester le routage avec vignette

Vous pouvez tester un itinéraire avec la vignette activée directement sur la [Live Map Waze](https://www.waze.com/live-map/directions?rp_subscription=vignette-switzerland).
