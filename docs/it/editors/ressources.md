# Risorse utili

Questa pagina raccoglie i link esterni e gli strumenti utili per editare la mappa Waze in Svizzera. In caso di dubbio, le risorse locali (Wiki svizzero, forum, Discord) hanno sempre la precedenza sull'improvvisazione.

## Wiki e Wazeopedia

Il Wiki di lingua tedesca è il riferimento per l'editing in Svizzera, poiché le traduzioni francesi possono accumulare ritardo.

- Wazeopedia Svizzera: https://www.waze.com/discuss/c/wazeopedia/switzerland-wazeopedia/5228
- Hub francofono (Informazioni in francese): https://www.waze.com/discuss/t/informations-en-francais/377254
- Hub germanofono (Hauptseite): https://www.waze.com/discuss/t/hauptseite/377253

::: note Lingue
L'hub francofono riunisce le pagine tradotte per la Svizzera romanda. La Hauptseite resta la pagina principale lato DACH.
:::

## Strumenti

### WME Switzerland Helper

Userscript Tampermonkey per l'editor. Aggiunge:

- livelli geografici svizzeri (confini, immagini aeree, ecc.),
- i dati dei trasporti pubblici (fermate SBB),
- una verifica dei nomi delle strade.

::: note Nota
Lo script si installa tramite un gestore di userscript come Tampermonkey.
:::
