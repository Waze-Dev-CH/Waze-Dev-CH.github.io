# Pedestrian paths

Waze is a car navigation app. Pedestrian paths are only mapped when they genuinely serve road navigation. The rest clutters the display with no benefit.

## When to include a pedestrian path

Two cases justify drawing a pedestrian path:

- it serves as a visual navigation aid for the driver, especially when the segment carries a well-known name
- it enables routing to an otherwise unreachable destination, by guiding the driver to the right access point when the destination can only be reached on foot

## When not to include

Not to map:

- sidewalks along roads
- hidden trails, invisible from the road and only spotted on aerial views
- internal paths within parks or private properties

::: important Important
A pedestrian path is only justified if it concretely improves the driver's routing or orientation. When in doubt, don't draw it.
:::
