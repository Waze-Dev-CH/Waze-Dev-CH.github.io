# Permanent hazards

This page presents the editing rules for permanent hazards in Switzerland.

::: important Reserved for L4 editors and above
Editing permanent hazards in Switzerland is reserved for editors of level L4 and above. Editors of a lower level must not create or modify these objects.
:::

## Framework

Permanent hazards warn drivers of a lasting risk linked to the road. Their editing in Switzerland follows specific rules, maintained by the community.

The complete guidelines have been updated and are published as a dedicated reference document, accessible via the Swiss forum topic.

::: note Reference document to consult
The supported hazard types and the detailed procedure are set out in the official document linked from the Swiss forum. This content is not reproduced here. Refer to the reference document before any editing.
:::

::: important Caution rule
When in doubt, do not edit the map based on assumptions. Ask the question on the Swiss forum or on the community Discord server.
:::
