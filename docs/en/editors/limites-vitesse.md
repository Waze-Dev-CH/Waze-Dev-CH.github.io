# Speed limits

This page covers entering speed limits on segments in Switzerland.

## General principle

A segment's speed limit is entered according to the official signage in force on the road. The value entered must correspond to the limit actually signposted on site.

Speed limit changes come up regularly on the Swiss forum (for example a change reported on the Schartenstrasse in Wettingen). In case of a change, verify the new signage before updating the segment.
