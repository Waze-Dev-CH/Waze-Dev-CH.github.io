# Useful resources

This page gathers the external links and tools useful for editing the Waze map in Switzerland. When in doubt, local resources (Swiss Wiki, forum, Discord) always take precedence over improvisation.

## Wikis and Wazeopedia

The German-language Wiki is the reference for editing in Switzerland, as the French translations may lag behind.

- Wazeopedia Switzerland: https://www.waze.com/discuss/c/wazeopedia/switzerland-wazeopedia/5228
- French-speaking hub (Information in French): https://www.waze.com/discuss/t/informations-en-francais/377254
- German-speaking hub (Hauptseite): https://www.waze.com/discuss/t/hauptseite/377253

::: note Languages
The French-speaking hub gathers the pages translated for French-speaking Switzerland. The Hauptseite remains the main page on the DACH side.
:::

## Tools

### WME Switzerland Helper

Tampermonkey userscript for the editor. It adds:

- Swiss geographic layers (boundaries, aerial imagery, etc.),
- public transport data (SBB stops),
- a street name check.

::: note Note
The script is installed via a userscript manager such as Tampermonkey.
:::
