# Swiss map legend

The map legend describes the segment types used in Switzerland and how they are displayed in the Waze editor (WME). Each road type has a specific use and consequences on routing.

::: note Under construction
The details of the segment types, their colours and their use cases are not yet covered here. For now this page is an introduction. The full content will be added from the Swiss Wiki.
:::

## What the legend is for

The choice of segment type directly influences guidance. A wrong type can send drivers onto unsuitable roads or, conversely, exclude them from valid routes. The legend serves as a common reference so that all editors name and classify roads in the same way.
